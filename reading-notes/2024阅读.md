# 书单

```dataview
table aliases as name, authors, startingTime, endTime, published, ISBN, "[douban]("+douban+")" as douban, "[goodreads]("+goodreads+")" as goodreads, "[neodb]("+neodb+")" as neodb
from "Langeweile/reading-notes/book/2024" or "Langeweile/reading-notes/comics/2024"
sort endTime desc
```

```base
filters:
  or:
    - file.path.startsWith("Langeweile/reading-notes/comics/2024")
    - file.path.startsWith("Langeweile/reading-notes/book/2024")
formulas:
  month: startingTime.format("YYYY-MM")
views:
  - type: table
    name: 表格
    groupBy:
      property: formula.month
      direction: DESC
    order:
      - file.name
      - aliases
      - authors
      - file.tags
      - startingTime
      - endTime
      - published
      - douban
      - goodreads
      - neodb
      - formula.month
    sort:
      - property: formula.month
        direction: DESC
      - property: startingTime
        direction: DESC

```
