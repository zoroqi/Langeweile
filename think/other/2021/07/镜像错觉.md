# 镜像错觉

坐电梯有很多人的时候, 经常看不见楼层, 只能通过电梯墙壁的镜像来知道当前的楼层. 但这里我多次看错楼层, 本身是12层, 但我看成了15层, 我就提现下电梯了, 我还需要爬几层.

```
 │ ─────┐  \  ┌───── │
 │      │  \  │      │
 │      │  \  │      │
 │ ┌────┘  \  └────┐ │
 │ │       \       │ │
 │ │       \       │ │
 │ └─────  \  ─────┘ │
```

在我看的时候, `2`翻转了成了`5`, 但`12`并没有翻转成`51`, 我潜意识理解成了`15`就提前下电梯了.

我猜测我的潜意识流程是这样的

1. 大楼总共26层, 但公司最高16层
2. 并且3~7层, 20~26是无法选择的
3. 当出现`5`大概率就是"15层"了
4. 墙壁中出现了`5`并没有反应过来是镜像的图案.
5. 人多赶紧下电梯
